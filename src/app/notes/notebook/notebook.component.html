Installation

  • Install Node and NPM
  • Install Angular CLI
  • Run ng new project-name
  • Go to project directory
  • Run ng serve
  • Adding bootstrap using cli: npm install --save bootstrap

Creating Component

  • Run ng g c component-name to create a component
  • When using angular cli for creating component it declares the component in module file by default.
    If created manually add it to declartions in NgModule file

Creating a module

  • Run ng g m module-name to create a module
  • To create a component inside a module use ng g c module-name/component-name

Data Binding

  • Property binding
  • Event binding
  • Two way data binding
